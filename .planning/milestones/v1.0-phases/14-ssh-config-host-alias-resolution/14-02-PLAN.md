  In config.go:

  1. Change LoadFile signature to: func LoadFile(dir string) (FileConfig, bool, error)
     - When os.IsNotExist: return FileConfig{}, false, nil
     - When read succeeds and YAML parses: return fc, true, nil
     - When read succeeds but YAML fails: return FileConfig{}, true, fmt.Errorf("parsing deploy.yaml: %w", err)
     - When os.ReadFile fails for any reason other than NotExist: return FileConfig{}, false, fmt.Errorf("reading deploy.yaml: %w", err)

       (A permission error on an existing file is unusual; false here means callers emit the not-found message variant, which is conservative and not misleading)

       - fileExisted=true:  fmt.Errorf("deploy.yaml: target.host is not set")
     This is exported (capital N) so callers in main.go can import and use it rather than inlining the strings, ensuring the exact message strings are defined in one place (testable in config package per D-15).
