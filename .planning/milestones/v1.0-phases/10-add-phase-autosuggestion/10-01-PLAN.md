  <implementation>
    Add ListHosts after the LookupHost function in sshconfig.go.

    Function signature: func ListHosts(configPath string) []string

    Implementation mirrors the LookupHost file-open and scanner loop skeleton.
    Inside the scanner loop: skip blank lines and comment lines; parse
    parts := strings.Fields(line); check keyword := strings.ToLower(parts[0]);
    when keyword == "host", iterate parts[1:] and append each pattern that does
    NOT contain '*' or '?' characters (use strings.ContainsAny(pattern, "*?") to
    detect wildcards). Return nil on file open error and on scanner.Err() != nil.
    Return the accumulated aliases slice (may be nil if no non-wildcard Host
    blocks found).

    The function does NOT need the Include, HostName, User, Port, or IdentityFile
    keyword cases — it only cares about "host" lines.

    Nolint comments to include: //nolint:gosec on os.Open (same rationale as
    LookupHost), //nolint:errcheck on defer f.Close().
