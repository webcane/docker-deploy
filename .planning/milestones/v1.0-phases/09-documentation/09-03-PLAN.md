    H2: "## Installation"
    Four subsections (D-16):

    H3: "### Option 1: Homebrew (macOS / Linux)"
    Two commands:
      brew tap webcane/docker-deploy
      brew install docker-deploy
    Note: Homebrew installs the binary to the Homebrew prefix. Add a symlink to the Docker CLI plugin directory:
      mkdir -p ~/.docker/cli-plugins
      ln -sf $(brew --prefix)/bin/docker-deploy ~/.docker/cli-plugins/docker-deploy

    H3: "### Option 2: Install script (macOS / Linux)"
    Single command:
      curl https://raw.githubusercontent.com/webcane/docker-deploy/main/install.sh | sh
    Note: Installs the latest release. To pin a version:
      INSTALL_VERSION=v1.0.0 curl https://raw.githubusercontent.com/webcane/docker-deploy/main/install.sh | sh

    H3: "### Option 3: Manual binary (GitHub Releases)"
    Steps: go to releases page, download the archive for your OS/arch (e.g. docker-deploy_linux_amd64.tar.gz), extract, move to ~/.docker/cli-plugins/, chmod +x.
    Provide the exact commands for linux/amd64 as an example (user substitutes their arch).

    H3: "### Option 4: go install"
    Command:
      GOBIN=~/.docker/cli-plugins go install github.com/webcane/docker-deploy/cmd/docker-deploy@latest
    Note: GOBIN must be set to ~/.docker/cli-plugins — Docker CLI plugins must live there, not in $GOPATH/bin (D-16 requirement).
