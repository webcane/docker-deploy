Update cmd/docker-deploy/main.go RunE to implement the --dry-run path. All other paths still return fmt.Errorf("deploy not implemented yet").

The RunE body when dryRun == true:

1. Determine projectName: base name of os.Getwd() result. If os.Getwd() fails, return the error.

2. Load deploy.yaml: call config.LoadFile(cwd) where cwd is the working directory string. If LoadFile returns an error, return it wrapped with fmt.Errorf("loading deploy.yaml: %w", err).

3. Resolve config: call config.Resolve(host, path, fileConfig, projectName). If Resolve returns an error, return it wrapped with fmt.Errorf("resolving config: %w", err).

4. Validate resolved config has a host: if resolved.Host.Hostname == "" return fmt.Errorf("no host configured: use --host flag or set target.host in deploy.yaml").

5. Build ssh.DialConfig from resolved config:
   - User: resolved.Host.User
   - Hostname: resolved.Host.Hostname
   - Port: resolved.Host.Port (default to 22 if 0)
   - Timeout: 10 * time.Second
   - Stdin: os.Stdin
   - Stdout: os.Stderr

6. Dial: call sshpkg.Dial(context.Background(), dialCfg) — import the internal/ssh package aliased as sshpkg to avoid conflict with golang.org/x/crypto/ssh. If error, print the error to os.Stderr and return it (exit non-zero satisfies D-13).

7. Defer client.Close().

8. Print dry-run summary to os.Stdout (D-12). Lines:
   - "Dry-run: connectivity check passed"
   - "  Host:        <user>@<hostname>:<port>"
   - "  Remote path: <resolved.Path>"
   - "  Auth method: ssh-agent / key file (whichever succeeded)"
   - "  Server:      <client.Conn.ServerVersion()>"
   - "  Status:      OK"

For the auth method line: check if $SSH_AUTH_SOCK is set — if so print "ssh-agent"; otherwise print "key file (~/.ssh/config)". This is a best-effort indicator, not authoritative.

Do not refactor the cobra command structure — keep plugin.Run() as the outermost wrapper. Add a runDryRun(host, path string, dockerCli command.Cli) error helper function if the RunE body grows unwieldy; keep the cobra plumbing in the anonymous function.
