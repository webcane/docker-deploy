Purpose: Operators get a familiar Docker-style configuration syntax that mirrors compose HEALTHCHECK; absent block means health polling is skipped, with defaults living in the global config file (per D-04).
Output: Updated `internal/config/config.go`, `internal/config/config_test.go`, and `cmd/docker-deploy/main.go`.
