## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-07 | Tampering | Replace-confirmation default N | mitigate | Default is No ([y/N]); only explicit "y" or "yes" proceeds. Accidental Enter does not trigger overwrite. Per D-09. |
| T-03-08 | Information Disclosure | --force in deploy.yaml | accept | force: true is an operator-managed config file on the developer's local machine. No remote exposure. |
| T-03-09 | Spoofing | SSH host during file upload | accept | SFTP wraps the same authenticated *gossh.Client from Dial(); knownhosts verification already enforced in Plan 02. No new trust surface. |
| T-03-10 | Information Disclosure | Per-file upload log to stdout | accept | File names are local paths chosen by the operator. No secrets in filenames. .env content is not printed, only the path. |
