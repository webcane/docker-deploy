# Quick Task 260523-n24: Fix CI Node.js 20 Deprecation Warnings

## Background

GitHub Actions CI is emitting deprecation warnings:
- `actions/checkout@v4` running on Node.js 20 (deprecated)
- `actions/setup-go@v5` running on Node.js 20 (deprecated)

Node.js 24 becomes mandatory June 2, 2026. Node.js 20 removed September 16, 2026.

## Task 1: Update ci.yml

**Files:** `.github/workflows/ci.yml`
**Action:** Add `FORCE_JAVASCRIPT_ACTIONS_TO_NODE24: "true"` env var to all jobs (`test`, `integration`)
**Verify:** Each job has the env var set
**Done:** No more Node.js 20 warnings for ci.yml jobs

## Task 2: Update release.yml

**Files:** `.github/workflows/release.yml`
**Action:** Add `FORCE_JAVASCRIPT_ACTIONS_TO_NODE24: "true"` env var to the `release` job
**Verify:** The release job has the env var set
**Done:** No more Node.js 20 warnings for release.yml job
