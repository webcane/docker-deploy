# Summary: Fix CI Node.js 20 Deprecation Warnings

## What was done

Added `FORCE_JAVASCRIPT_ACTIONS_TO_NODE24: "true"` env var to all job definitions in both GitHub Actions workflow files:

- `.github/workflows/ci.yml` — added to `test` and `integration` jobs
- `.github/workflows/release.yml` — added to `release` job

## Why this approach

The env var opts into Node.js 24 for action runners immediately, eliminating the deprecation warnings for `actions/checkout@v4` and `actions/setup-go@v5`. This is the GitHub-recommended approach prior to the forced migration (June 2, 2026).
